Detalle del Proyecto Laberinto Python

Este proyecto consiste en generar y resolver laberintos de forma aleatoria usando el lenguaje de programación Python. Se utiliza el algoritmo de recorrido en profundidad para crear el laberinto y un algoritmo de prueba y error para encontrar la salida. El laberinto se representa como una matriz de caracteres, donde las paredes son 'X' y los espacios vacíos son ' '. La entrada está en la esquina superior izquierda y la salida en la inferior derecha. El programa muestra el laberinto por pantalla y la secuencia de movimientos necesarios para resolverlo.
